# AWS EC2 Web Server Deployment

## Project Overview

This project demonstrates how to host a static website on an Amazon EC2 instance using the Apache Web Server.

## Technologies Used

- AWS EC2
- Amazon Linux
- Apache HTTP Server
- Linux
- HTML
- SSH

## Steps Performed

- Launched an Amazon EC2 instance
- Configured Security Groups
- Connected using SSH
- Installed Apache
- Started and enabled Apache
- Deployed a custom index.html page
- Accessed the website using the EC2 Public IP
